=========================== Concepts ===========================

Active Move - A Move that may take additional time to complete.

Passive Move - A Move that cannot voluntarily take longer to complete, but can remain active as long as third-party sources continue to select it.
 
=========================== Naming ===========================

All moves are classes with naming rule below.

Standard F prefix by Unreal.
-----------------------------------
PM - Passive Move.

or

AM - Active Move.
-----------------------------------
"_" symbol.
-----------------------------------
Name of Move itself. Example: "Step"

=========================== Name Examples ===========================

Class file name example: AM_Step.h
Class name example: FAM_Step
